Suggested Demo Flow

This sequence works well for a short presentation or oral examination demo.

  • Open Swagger UI at /docs to show the documented API surface.
  • Call GET /gold/prices/latest to show live market data retrieval.
  • Call GET /gold/analytics/summary and GET /gold/analytics/regime to demonstrate analytical value beyond CRUD.
  • Show the protected flow: register -> login -> create alert -> list alerts.
  • Use this visual report as a compact summary that the project is a gold intelligence API, not only a basic CRUD exercise.
